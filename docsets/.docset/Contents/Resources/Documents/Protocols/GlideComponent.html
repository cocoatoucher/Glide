GlideComponent

public protocol GlideComponent

Basic component protocol that should be adopted by all components for full compatibility of features.

  • componentPriority

    Priority of the component within other components of its entity. Components with priority of lower values will be updated before components with priority of higher values. Default value is 0.

    Declaration

    Swift

    static var componentPriority: Int { get }
  • transform

    Convenience reference to TransformNodeComponent of this component’s entity.

    Declaration

    Swift

    var transform: TransformNodeComponent? { get }
  • scene

    Convenience reference to the scene of this component’s entity.

    Declaration

    Swift

    var scene: GlideScene? { get }
  • currentTime

    Convenience reference to the current time of the scene of this component’s entity.

    Declaration

    Swift

    var currentTime: TimeInterval? { get }
  • start()

    Called in either of those two situations:

    • Component is just added to an entity with a non empty scene.
    • Component is already a member of an entity’s component and that entity’s transform is just added to a scene. Don’t call this method directly.

    Declaration

    Swift

    func start()
  • willUpdate(deltaTime:)

    Called prior to calling update(deltaTime:) of GKComponent. Don’t call this method directly.

    Declaration

    Swift

    func willUpdate(deltaTime seconds: TimeInterval)

    Parameters

    seconds

    Seconds passed since the last frame.
  • didUpdate(deltaTime:)

    Called after calling update(deltaTime:) of GKComponent. Don’t call this method directly.

    Declaration

    Swift

    func didUpdate(deltaTime seconds: TimeInterval)

    Parameters

    seconds

    Seconds passed since the last frame.
  • updateAfterCameraUpdate(deltaTime:cameraComponent:)

    Called after the camera entity of the entity’s scene is updated. Don’t call this method directly.

    Declaration

    Swift

    func updateAfterCameraUpdate(deltaTime seconds: TimeInterval, cameraComponent: CameraComponent)

    Parameters

    seconds

    Seconds passed since the last frame.
    cameraComponent

    Camera component of the scene’s camera entity.
